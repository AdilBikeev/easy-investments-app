# Конфигурация приложения

1. Откройте консоль в папке проекта "*\Stock.API.csproj"
2. Возьмите токен авторизации с [личного кабинета Tinkoff](https://www.tinkoff.ru/invest/)
	2.1. Перейдите в настройки
	2.2.Проверьте, что функция “Подтверждение сделок кодом” отключена
	2.3.Выпустите токен для торговли на бирже и режима “песочницы” (sandbox)
	2.4.Скопируйте токен и сохраните, токен отображается только один раз, просмотреть его позже не получится, тем не менее вы можете выпускать неограниченное количество токенов
3. Добавьте токен в user-secrets
	```csharp
	dotnet user-secrets set "AccessToken" "<token>"
	```

Подробнее про user-secrets можно почитать [тут](https://docs.microsoft.com/ru-ru/aspnet/core/security/app-secrets?view=aspnetcore-3.1&tabs=windows)
